Unofficial Simplified Chinese Translation for Clickteam Fusion 2.5

Clickteam Fusion 2.5非官方汉化

### 使用方法

1. 复制zh-cn文件夹至Clickteam Fusion 2.5安装路径Languages文件夹下
2. 运行Fusion，依次打开Tools→Preferences→General，选择Languages为中文(中国)
3. 重新启动Fusion
